# Proposal: Меню v2 — Клиентский флоу + Админ-панель + 3x-ui подключение

## Контекст

Бот задеплоен (@Test123PicoBot, vpn-relay 172.86.95.214, polling mode). PR #3 смёржен в main.
Текущая проблема: большинство кнопок главного меню не работают или ведут не туда.
Вся админ-панель выдаёт «Нет активных серверов» — бот не подключён к 3x-ui.
Цель: довести бота до MVP с рабочим клиентским флоу, рабочей админкой и живым VPN.

---

## Роли

| Роль | Боль сейчас | Изменение |
|------|-------------|-----------|
| User | Большинство кнопок не работает; нет автотриала на /start | Все кнопки рабочие; триал выдаётся автоматически |
| Admin | Вся админ-панель — «нет серверов»; нет удобного confirm/reject | Все кнопки работают; список pending + confirm/reject оплат |

---

## Решение

1. Перестроить главное меню клиента — только нужные кнопки, все рабочие.
2. Починить подключение к 3x-ui через /admin_server.
3. Скрыть ненужные методы оплаты (оставить только СБП + TON).
4. Починить все кнопки Admin Tools.
5. Добавить поддержку нескольких admin_id.

---

## Scope

**In scope:**
- Главное меню клиента: 6 кнопок (см. флоу ниже)
- /start → авто-регистрация + авто-триал 3 дня (без клика)
- Раздел «Заполнить профиль» — FSM device+email → +4 дня
- Раздел «Моя подписка» — статус, история, кнопка «Продлить»
- Раздел «Оплатить» — только СБП + TON, скрыть Stars/YooKassa/etc.
- Раздел «Рефералы» — ссылка, счётчик, бонусные дни
- Раздел «Написать нам» — FSM ввод текста → forward на SUPPORT_ID
- Раздел «Инструкции» — кнопка-заглушка (content будет в отдельной спеке)
- Admin Tools: все кнопки (/admin_server, /admin_stats, /admin_users, /admin_payments, /admin_promos, /admin_notify, /admin_backup, /admin_maintenance)
- /admin_payments: список PENDING-оплат + confirm ✅ / reject ❌
- Подключение 3x-ui: диагностика «нет активных серверов» + добавление сервера через /admin_server
- Multi-admin: поддержка нескольких tg_id в BOT_ADMINS

**Out of scope:**
- Содержимое раздела «Инструкции» (текст, QR, ссылки на приложения)
- Автопроверка оплаты через API банка/блокчейна
- YooKassa, Cryptomus, YooMoney — не показывать, не удалять из кода
- Промокоды, частичная оплата
- Многоуровневая admin-иерархия

## Клиентский флоу (happy path)

```
/start
  └─ User не существует → create User(vpn_id=UUID)
  └─ _try_auto_gift_trial() → триал 3 дня выдан автоматически
  └─ Главное меню:
       ┌─────────────────────────────┐
       │  📋 Заполнить профиль (+4д) │  ← FSM: device → email → bonus days
       │  📡 Моя подписка            │  ← статус + история + Продлить
       │  💳 Оплатить                │  ← СБП / TON
       │  👥 Пригласить друга        │  ← реф. ссылка + счётчик
       │  💬 Написать нам            │  ← FSM → forward на SUPPORT_ID
       │  📖 Инструкции              │  ← заглушка «скоро»
       └─────────────────────────────┘

Оплата (СБП):
  Выбрать СБП → реквизиты (+7 903 918 90 74, Цифра банк, 100₽) → «Я оплатил»
    └─ нет pending → Transaction(PENDING) + подписка +30д оптимистично
    └─ есть pending → показать блокировку

Оплата (TON):
  Выбрать TON → реквизиты (@Tapac1990 / адрес кошелька, 1 USDT) → «Я оплатил»
    └─ аналогично СБП

Admin:
  /admin_payments → список PENDING:
    [username | метод | дата | сумма] [✅ Подтвердить] [❌ Отклонить]
    └─ ✅ → COMPLETED + реф. хук + уведомление юзеру
    └─ ❌ → REJECTED + вычет 30д (не в прошлое) + уведомление юзеру
    └─ двойной клик → игнор
```
